# TCC_UspEsalq
Esse repositório de códigos faz parte do Trabalho de Conclusão de Curso [TCC] do MBA de Ciência de Dados da USB Esalq. Autoria :  Iago Valagna Pelisson linkedin : https://www.linkedin.com/in/iago-pelisson-a5280a142/

O objetivo desse projeto foi desenvolver e validar uma base de dados histórica do Campeonato Brasileiro Série A (2013–2023), aplicando técnicas de ciência de dados e visualização interativa para analisar desempenho esportivo, perfil financeiro dos elencos e engajamento das torcidas.
